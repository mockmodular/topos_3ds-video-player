# 2026-03-27 05:45 — 字幕代码全量删除

## 改动范围

从 `video_player.c`、`decoder.c`、`decoder.h`、`media_types.h`、`decoder_types.h` 完全删除字幕（subtitle）相关代码。

## 删除内容汇总

### video_player.c（~312行削减）
- `#define SUBTITLE_BUFFERS` 宏
- `HID_S_TRACK_*`、`HID_SE0_S_TRACK_*`、`HID_SE1_SUBTITLE_*` 全部 HID 宏
- `DECODE_THREAD_CHANGE_SUBTITLE_TRACK_REQUEST` 枚举值
- `MSG_MOVE_MODE_ENABLE_SUBTITLE`、`MSG_SUBTITLE_TRACK_DESCRIPTION`、`MSG_SUBTITLE_TRACK`、`MSG_DISABLE_SUBTITLE` 枚举值
- `MOVE_SUBTITLE` 枚举值
- 播放器状态结构体中所有字幕字段（`disable_subtitle`、`num_of_subtitle_tracks`、`selected_subtitle_track`、`subtitle_index`、`subtitle_x/y_offset`、`subtitle_zoom`、`subtitle_info[]`、`subtitle_data[]`、`subtitle_image[]`、`is_selecting_subtitle_track`）
- 所有字幕按钮（`subtitle_track_ok_button`、`subtitle_track_button[]`、`select_subtitle_track_button`、`disable_subtitle_button`）
- `Vid_init_subtitle_data()` 声明及实现
- 所有字幕 UI 绘制、HID 输入处理、轨道选择弹窗
- 字幕 zoom/offset 滚动逻辑（`subtitle_size_changes` 变量块）
- 解码线程中 `DECODE_THREAD_CHANGE_SUBTITLE_TRACK_REQUEST` case
- `else if(type == MEDIA_PACKET_TYPE_SUBTITLE)` packet 分发分支
- 设置读写/log 中的 `disable_subtitle` 字段

### decoder.c
- `util_subtitle_decoder_*` 所有静态变量（init/packet_ready/cache_packet_ready/stream_num/packet/cache_packet/raw_data/context/codec 数组）
- `Util_decoder_subtitle_exit()` 静态函数
- `Util_decoder_subtitle_init()` 函数
- `Util_decoder_subtitle_get_info()` 函数
- `Util_decoder_ready_subtitle_packet()` 函数
- `Util_decoder_skip_subtitle_packet()` 函数
- `Util_decoder_open_file()` 中 subtitle 流检测分支、`subtitle_index` 变量

### decoder.h
- `Util_decoder_subtitle_init/get_info/ready_packet/skip_packet` 声明
- `Util_decoder_open_file()` 签名更新：移除 `num_of_subtitle_tracks` 参数

### media_types.h / decoder_types.h
- `Media_s_info` 结构体（字幕轨道信息）
- `Media_s_data` 结构体（字幕数据）
- `DEF_DECODER_MAX_SUBTITLE_TRACKS` 常量
- `MEDIA_PACKET_TYPE_SUBTITLE` 枚举值

## API 变更

`Util_decoder_open_file()` 签名变化：
```c
// 改前
uint32_t Util_decoder_open_file(const char* path, uint8_t* num_of_audio_tracks, uint8_t* num_of_video_tracks, uint8_t* num_of_subtitle_tracks, uint8_t session);
// 改后
uint32_t Util_decoder_open_file(const char* path, uint8_t* num_of_audio_tracks, uint8_t* num_of_video_tracks, uint8_t session);
```
所有调用点（video_player.c 中）已同步更新。
