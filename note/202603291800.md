# 2026-03-29 打开新视频全屏/下屏状态

## 现象
刚打开新视频时像留在 UI 或非全屏/下屏息屏逻辑，需点下屏才正常。

## 原因（推断）
1. `if(!vid_player.is_full_screen)` 导致连续播第二条时若已为全屏则跳过 `Vid_enter_full_screen`，`turn_off_bottom_screen_count` 等与 Sem 下屏开关不同步。
2. `Vid_enter_full_screen` 未恢复 `is_bottom_lcd_on`/下屏亮度，若上次全屏倒计时已关下屏，新文件可能仍异常。

## 修改
- `Vid_enter_full_screen`：进入时 `Sem_set_config` 打开下屏并同步上下亮度（与 `Vid_exit_full_screen` 一致）。
- 打开有视频轨文件：去掉 `if(!vid_player.is_full_screen)`，始终 `Vid_enter_full_screen(0)`，并 `auto_full_screen_count = 0`。
