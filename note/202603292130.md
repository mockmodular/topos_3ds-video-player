# 2026-03-29 21:30 — 打开新视频不触发全屏切换

## 问题
打开新视频时自动进入全屏模式（亮屏的息屏/全屏状态），下屏亮但 X 键无效，需触碰下屏退出全屏后才能按 X 打开文件菜单。

## 修改
**文件**：`source/video_player.c` ~3771

删除以下两行：
```c
vid_player.auto_full_screen_count = 0;
Vid_enter_full_screen(0);
```

顺带更新注释：`//Enter full-screen mode if file has video tracks.` → `//Fit video to screen if file has video tracks.`

## 结果
打开新视频不触发任何全屏/UI 模式切换，原来什么状态保持不变。
