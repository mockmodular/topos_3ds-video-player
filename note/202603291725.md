# 2026-03-29 17:25 — CPU占用率显示不可见 修复

## 问题
sem菜单 Advanced → CPU占用显示 ON/OFF 开关打开后，下屏无任何显示。

## 根因
`source/system/util/cpu_usage.c:152`：
```c
Draw_with_background_c(msg_cache, 360, 90, ...)
```
`DRAW_X_ALIGN_RIGHT` 时 x 是 box 左边界，box 宽=100，范围 360~460，3DS 下屏仅 320px 宽，完全超出屏幕不可见。

## 修复
`x=360` → `x=220`，box 范围变为 220~320，贴下屏右侧显示。

## 文件
- `source/system/util/cpu_usage.c:152`
